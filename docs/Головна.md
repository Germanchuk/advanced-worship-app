# Пісні
- не можуть належати юзеру
- можуть належати групі
- можуть належати загальній базі
- можуть належати списку

як шерити пісні між групами, юзерами?

# Шаблон - дефолтний варіант пісні в загальній базі, який може скопіювати будь-хто собі, щоб кастомізувати.


Способи вводу пісень:
- білдер
- копі-паста парсер
- парсер холікордс
